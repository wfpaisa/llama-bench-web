# PrimeNG Developer Guidelines

- When is required to generate components, follow the PrimeNG components documentation.

# List of PrimeNG components documentation

- [Angular Accordion Component](https://primeng.org/llms/components/accordion.md): Accordion groups a collection of contents in tabs.
- [Angular Animate On Scroll Directive](https://primeng.org/llms/components/animateonscroll.md): AnimateOnScroll is used to apply animations to elements when entering or leaving the viewport during scrolling.
- [Angular AutoComplete Component](https://primeng.org/llms/components/autocomplete.md): AutoComplete is an input component that provides real-time suggestions when being typed.
- [Angular AutoFocus Directive](https://primeng.org/llms/components/autofocus.md): AutoFocus manages focus on focusable element on load.
- [Angular Avatar Component](https://primeng.org/llms/components/avatar.md): Avatar represents people using icons, labels and images.
- [Angular Badge Component](https://primeng.org/llms/components/badge.md): Badge is a small status indicator for another element.
- [Angular BlockUI Component](https://primeng.org/llms/components/blockui.md): BlockUI can either block other components or the whole page.
- [Angular Breadcrumb Component](https://primeng.org/llms/components/breadcrumb.md): Breadcrumb provides contextual information about page hierarchy.
- [Angular Button Component](https://primeng.org/llms/components/button.md): Button is an extension to standard button element with icons and theming.
- [Angular Card Component](https://primeng.org/llms/components/card.md): Card is a flexible container component.
- [Angular Carousel Component](https://primeng.org/llms/components/carousel.md): Carousel is a content slider featuring various customization options.
- [Angular CascadeSelect Component](https://primeng.org/llms/components/cascadeselect.md): CascadeSelect displays a nested structure of options.
- [Angular Chart Component](https://primeng.org/llms/components/chart.md): Chart components are based on Charts.js 3.3.2+, an open source HTML5 based charting library.
- [Angular Checkbox Component](https://primeng.org/llms/components/checkbox.md): Checkbox is an extension to standard checkbox element with theming.
- [Angular Chip Component](https://primeng.org/llms/components/chip.md): Chip represents entities using icons, labels and images.
- [Angular ColorPicker Component](https://primeng.org/llms/components/colorpicker.md): ColorPicker is an input component to select a color.
- [Angular ConfirmDialog Component](https://primeng.org/llms/components/confirmdialog.md): ConfirmDialog is backed by a service utilizing Observables to display confirmation windows easily that can be shared by multiple actions on the same component.
- [Angular ConfirmPopup Component](https://primeng.org/llms/components/confirmpopup.md): ConfirmPopup displays a confirmation overlay displayed relatively to its target.
- [Angular ContextMenu Component](https://primeng.org/llms/components/contextmenu.md): ContextMenu displays an overlay menu on right click of its target.
- [Angular DataView Component](https://primeng.org/llms/components/dataview.md): DataView displays data in grid grid-cols-12 gap-4 or list layout with pagination and sorting features.
- [Angular DatePicker Component](https://primeng.org/llms/components/datepicker.md): DatePicker is an input component to select a date.
- [Angular Dialog Component](https://primeng.org/llms/components/dialog.md): Dialog is a container to display content in an overlay window.
- [Angular Divider Component](https://primeng.org/llms/components/divider.md): Divider is used to separate contents.
- [Angular Dock Component](https://primeng.org/llms/components/dock.md): Dock is a navigation component consisting of menuitems.
- [Angular Drag and Drop Component](https://primeng.org/llms/components/dragdrop.md): pDraggable and pDroppable directives apply drag-drop behaviors to any element.
- [Angular Drawer Component](https://primeng.org/llms/components/drawer.md): Drawer is a container component displayed as an overlay.
- [Angular Dynamic Dialog Component](https://primeng.org/llms/components/dynamicdialog.md): Dialogs can be created dynamically with any component as the content using a DialogService.
- [Angular Editor Component](https://primeng.org/llms/components/editor.md): Editor is rich text editor component based on Quill.
- [Angular Fieldset Component](https://primeng.org/llms/components/fieldset.md): Fieldset is a grouping component with a content toggle feature.
- [Angular FileUpload Component](https://primeng.org/llms/components/fileupload.md): FileUpload is an advanced uploader with dragdrop support, multi file uploads, auto uploading, progress tracking and validations.
- [Angular Float Label Component](https://primeng.org/llms/components/floatlabel.md): FloatLabel appears on top of the input field when focused.
- [Angular Fluid Component](https://primeng.org/llms/components/fluid.md): Fluid is a layout component to make descendant components span full width of their container.
- [Angular Focus Trap Component](https://primeng.org/llms/components/focustrap.md): Focus Trap keeps focus within a certain DOM element while tabbing.
- [Angular Gallery Component](https://primeng.org/llms/components/galleria.md): Galleria is an advanced content gallery component.
- [Angular IconField Component](https://primeng.org/llms/components/iconfield.md): IconField wraps an input and an icon.
- [Angular Ifta Label Component](https://primeng.org/llms/components/iftalabel.md): IftaLabel is used to create infield top aligned labels.
- [Angular Image Component](https://primeng.org/llms/components/image.md): Displays an image with preview and tranformation options.
- [Angular ImageCompare Component](https://primeng.org/llms/components/imagecompare.md): Compare two images side by side with a slider.
- [Angular Inplace Component](https://primeng.org/llms/components/inplace.md): Inplace provides an easy to do editing and display at the same time where clicking the output displays the actual content.
- [Angular InputGroup Component](https://primeng.org/llms/components/inputgroup.md): Text, icon, buttons and other content can be grouped next to an input.
- [Angular InputMask Component](https://primeng.org/llms/components/inputmask.md): InputMask component is used to enter input in a certain format such as numeric, date, currency and phone.
- [Angular InputNumber Component](https://primeng.org/llms/components/inputnumber.md): InputNumber is an input component to provide numerical input.
- [Angular InputText Component](https://primeng.org/llms/components/inputtext.md): InputText is an extension to standard input element with theming and keyfiltering.
- [Angular KeyFilter Component](https://primeng.org/llms/components/keyfilter.md): KeyFilter is a directive to restrict individual key strokes. In order to restrict the whole input, use InputNumber or InputMask instead.
- [Angular Knob Component](https://primeng.org/llms/components/knob.md): Knob is a form component to define number inputs with a dial.
- [Angular Listbox Component](https://primeng.org/llms/components/listbox.md): Listbox is used to select one or more values from a list of items.
- [Angular MegaMenu Component](https://primeng.org/llms/components/megamenu.md): MegaMenu is navigation component that displays submenus together.
- [Angular Menu Component](https://primeng.org/llms/components/menu.md): Menu is a navigation / command component that supports dynamic and static positioning.
- [Angular Menubar Component](https://primeng.org/llms/components/menubar.md): Menubar is a horizontal menu component.
- [Angular Message Component](https://primeng.org/llms/components/message.md): Message component is used to display inline messages.
- [Angular MeterGroup Component](https://primeng.org/llms/components/metergroup.md): MeterGroup displays scalar measurements within a known range.
- [Angular MultiSelect Component](https://primeng.org/llms/components/multiselect.md): MultiSelect is used to select multiple items from a collection.
- [Angular OrderList Component](https://primeng.org/llms/components/orderlist.md): OrderList is used to sort a collection.
- [Angular Organization Chart Component](https://primeng.org/llms/components/organizationchart.md): OrganizationChart visualizes hierarchical organization data.
- [Angular Otp Input Component](https://primeng.org/llms/components/inputotp.md): Input Otp is used to enter one time passwords.
- [Angular Paginator Component](https://primeng.org/llms/components/paginator.md): Paginator displays data in paged format and provides navigation between pages.
- [Angular Panel Component](https://primeng.org/llms/components/panel.md): Panel is a container component with an optional content toggle feature.
- [Angular PanelMenu Component](https://primeng.org/llms/components/panelmenu.md): PanelMenu is a hybrid of Accordion and Tree components.
- [Angular Password Component](https://primeng.org/llms/components/password.md): Password displays strength indicator for password fields.
- [Angular PickList Component](https://primeng.org/llms/components/picklist.md): PickList is used to reorder items between different lists.
- [Angular Popover Component](https://primeng.org/llms/components/popover.md): Popover is a container component that can overlay other components on page.
- [Angular ProgressBar Component](https://primeng.org/llms/components/progressbar.md): ProgressBar is a process status indicator.
- [Angular ProgressSpinner Component](https://primeng.org/llms/components/progressspinner.md): ProgressSpinner is a process status indicator.
- [Angular RadioButton Component](https://primeng.org/llms/components/radiobutton.md): RadioButton is an extension to standard radio button element with theming.
- [Angular Rating Component](https://primeng.org/llms/components/rating.md): Rating component is a star based selection input.
- [Angular Ripple Component](https://primeng.org/llms/components/ripple.md): Ripple directive adds ripple effect to the host element.
- [Angular Scroll Panel Component](https://primeng.org/llms/components/scrollpanel.md): ScrollPanel is a cross browser, lightweight and skinnable alternative to native browser scrollbar.
- [Angular Scroll Top Component](https://primeng.org/llms/components/scrolltop.md): ScrollTop gets displayed after a certain scroll position and used to navigates to the top of the page quickly.
- [Angular Select Component](https://primeng.org/llms/components/select.md): Select is used to choose an item from a collection of options.
- [Angular SelectButton Component](https://primeng.org/llms/components/selectbutton.md): SelectButton is used to choose single or multiple items from a list using buttons.
- [Angular Skeleton Component](https://primeng.org/llms/components/skeleton.md): Skeleton is a placeholder to display instead of the actual content.
- [Angular Slider Component](https://primeng.org/llms/components/slider.md): Slider is a component to provide input with a drag handle.
- [Angular Speed Dial Component](https://primeng.org/llms/components/speeddial.md): SpeedDial is a floating button with a popup menu.
- [Angular SplitButton Component](https://primeng.org/llms/components/splitbutton.md): SplitButton groups a set of commands in an overlay with a default action item.
- [Angular Splitter Component](https://primeng.org/llms/components/splitter.md): Splitter is utilized to separate and resize panels.
- [Angular Stepper Component](https://primeng.org/llms/components/stepper.md): The Stepper component displays a wizard-like workflow by guiding users through the multi-step progression.
- [Angular StyleClass Component](https://primeng.org/llms/components/styleclass.md): StyleClass manages css classes declaratively to during enter/leave animations or just to toggle classes on an element.
- [Angular Table Component](https://primeng.org/llms/components/table.md): Table displays data in tabular format.
- [Angular Tabs Component](https://primeng.org/llms/components/tabs.md): Tabs is a container component to group content with tabs.
- [Angular Tag Component](https://primeng.org/llms/components/tag.md): Tag component is used to categorize content.

- [Angular Textarea Component](https://primeng.org/llms/components/textarea.md): Textarea adds styling and autoResize functionality to standard textarea element.
- [Angular TieredMenu Component](https://primeng.org/llms/components/tieredmenu.md): TieredMenu displays submenus in nested overlays.
- [Angular Timeline Component](https://primeng.org/llms/components/timeline.md): Timeline visualizes a series of chained events.
- [Angular Toast Component](https://primeng.org/llms/components/toast.md): Toast is used to display messages in an overlay.
- [Angular ToggleButton Component](https://primeng.org/llms/components/togglebutton.md): ToggleButton is used to select a boolean value using a button.
- [Angular ToggleSwitch Component](https://primeng.org/llms/components/toggleswitch.md): ToggleSwitch is used to select a boolean value.
- [Angular Toolbar Component](https://primeng.org/llms/components/toolbar.md): Toolbar is a grouping component for buttons and other content.
- [Angular Tooltip Component](https://primeng.org/llms/components/tooltip.md): Tooltip directive provides advisory information for a component. Tooltip is integrated within various PrimeNG components.
- [Angular Tree Component](https://primeng.org/llms/components/tree.md): Tree is used to display hierarchical data.
- [Angular TreeSelect Component](https://primeng.org/llms/components/treeselect.md): TreeSelect is a form component to choose from hierarchical data.
- [Angular TreeTable Component](https://primeng.org/llms/components/treetable.md): TreeTable is used to display hierarchical data in tabular format.
- [Angular Virtual Scroller Component](https://primeng.org/llms/components/scroller.md): VirtualScroller is a performance-approach to handle huge data efficiently.
- [Overlay API - PrimeNG](https://primeng.org/llms/components/overlay.md): This API allows overlay components to be controlled from the PrimeNG. In this way, all overlay components in the application can have the same behavior.
